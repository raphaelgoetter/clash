# Images du jeu Frame

Placer ici les images des frames de films, au format `.webp`, nommées avec un
index sur 3 chiffres correspondant au champ `"image"` de `frames.json` :

```
001.webp
002.webp
003.webp
...
```

Servies en statique par Vite/Vercel à l'URL
`https://trustroyale.vercel.app/frames/images/NNN.webp`.
